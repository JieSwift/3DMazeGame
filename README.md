# 伪3D迷宫游戏 #
> 一个3D迷宫游戏，玩家操作通过操作人物方向走出迷宫
> W+S+A+D操作迷宫游戏方向
> Q+E操作游戏视角

## 游戏采用技术 ##
> 使用canvas制作伪3D迷宫游戏。
> 使用原生JavaScript来制作游戏。
